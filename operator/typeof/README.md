# operator typeof

digunakan untuk mengecek sebuah tipe data dari sebuah variabel,
hasil dari operator ini adalah string tipe datanya

|type|hasil|
|----|-----|
|`undefined`|"undefined"|
|`null`|"object"|
|`boolean`|"boolean"|
|`number`|"number"|
|`bigint`|"bigint"|
|`string`|"string"|
|`symbol`|"symbol"|
|`function`|"function"|
|lainnya|"object"|
