operator logika adalah operasi untuk dua buah data boolean, hasil dari operasi ini adalah boolean lagi.

|operator|keterangan|
|--------|----------|
|&&|and|
|\|\||atau|
|!|kebalikan|

### operator &&

|nilai 1|operator|nilai 2|hasil|
|-------|--------|-------|-----|
|true|&&|true|true|
|true|&&|false|false|
|false|&&|false|false|
|false|&&|true|false|

### operator ||

|nilai 1|operator|nilai 2|hasil|
|-------|--------|-------|-----|
|true|\|\\||true|true|
|true|\|\\||false|true|
|false|\|\\||false|false|
|false|\|\\||true|true|

## operator !

|operator|nilai 2|hasil|
|--------|-------|-----|
|!|true|false|
|!|false|true|
